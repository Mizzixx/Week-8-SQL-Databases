# Week 8 – SQL Database & CSV Integration Project

This project demonstrates a Python-based SQLite database application that imports CSV files, supports user authentication, and manages incidents.  
It’s designed for learning purposes and to show how to connect CSV data with a database while performing basic CRUD operations.

---

## Features

- **Database Setup**: Automatically creates SQLite database and tables  
- **CSV Import**: Reads CSV files from the `DATA/` folder and maps them to database tables  
- **User Authentication**: Register and login users with hashed passwords using `bcrypt`  
- **Incident Management**: Insert new incidents and view all incidents  
- **Debug Logging**: Shows CSV columns, first rows, mapping info, and database summary
